Mac й…ҚзҪ®ж•ҷзЁӢ-ж—ҘеёёзҜҮ

еҸ‘иЎЁдәҺ 2018-12-08 | жӣҙж–°дәҺ 2024-03-19 | еҲҶзұ»дәҺ ToolsDaily

д»Ҡе№ҙз»ҲдәҺеңЁжҺЁеҮә 2018 ж¬ҫ MBP ж—¶пјҢзңӢеҲ°еҚҮзә§дәҶ CPUпјҢжҲ‘е°ұжһңж–ӯдёӢжүӢгҖҢжӢ”иҚүгҖҚгҖӮжң¬ж–Үи®°еҪ•дҪҝз”Ё Mac зҡ„дёҖдәӣй…ҚзҪ®пјҢдјҡй•ҝжңҹжӣҙж–°гҖӮ

дёәдәҶжҺ§еҲ¶ж–Үз« зҡ„зҜҮе№…пјҢжҲ‘е°Ҷ Mac дҪҝз”Ёй…ҚзҪ®еҲҶжҲҗдәҶдёӨзҜҮпјҡ

зі»з»ҹй…ҚзҪ®

и®°еҪ•дёҖдәӣ Mac зі»з»ҹй…ҚзҪ®ж–№йқўзҡ„жҠҖе·§пјҢи®©дҪҝз”Ёиө·жқҘжӣҙж–№дҫҝгҖӮ

жҢүй”®ж•ҙзҗҶ

  • EnterпјҡйҖүдёӯдёҖдёӘж–Үд»¶жҢүдёӢ EnterпјҢзӣҙжҺҘж”№еҗҚ
  • Ctrl еӨҡз”ЁдәҺе‘Ҫд»ӨиЎҢеҝ«жҚ·й”®
  • Command еӨҡз”ЁдәҺеӣҫеҪўеҢ–еҝ«жҚ·й”®
  • Option й”®жңүзӮ№еғҸвҖңжҚўжҢЎвҖқй”®пјҢжҜ”еҰӮжҢүдҪҸOptionй”®еҶҚеҺ»зӮ№еҮ»е·ҰдёҠи§’зҡ„иӢ№жһңеӣҫж ҮпјҢдјҡеӨҡжҳҫзӨәвҖңзі»з»ҹдҝЎжҒҜвҖқпјҢеҶҚжҜ”еҰӮжғіејәеҲ¶йҖҖеҮәжҹҗдёӘиҪҜд»¶ж—¶пјҢжҢүдҪҸOptionй”®пјҢдјҡзңӢеҲ°вҖңејәеҲ¶йҖҖеҮәвҖқзҡ„йҖүйЎ№гҖӮ
  • йҖүдёӯжЎҢйқўзҡ„жүҖжңүйңҖеҪ’жЎЈиө„ж–ҷпјҢCommand+Shift+nпјҢдјҡе°Ҷж–Үд»¶еҪ’жЎЈеҲ°дёҖдёӘж–Үд»¶еӨ№дёӢ
  • и°ғеәҰдёӯеҝғи®ҫзҪ®пјҡеҸі CtrlпјҢжҳҫзӨәжЎҢйқў
  • и§ҰеҸ‘и§’

  • иҫ…еҠ©еҠҹиғҪ-йј ж ҮдёҺжҺ§еҲ¶жқҝ-и§ҰжҺ§жқҝйҖүйЎ№-еҗҜз”ЁжӢ–з§»
  • Option+Command+v зұ»дјјеүӘеҲҮзҡ„еҠҹиғҪпјҢе°ұжҳҜзІҳиҙҙеҗҺпјҢеҺҹжқҘж–Үд»¶дјҡиў«еүӘеҲҮиҝҮжқҘгҖӮеӨҚеҲ¶ж–Үд»¶еҗҺпјҢйҷӨдәҶз”Ёеҝ«жҚ·й”®пјҢиҝҳеҸҜд»ҘеңЁж–Үд»¶еӨ№з©әзҷҪеӨ„еҸій”®пјҢз„¶еҗҺеҶҚжҢүдҪҸ гҖҢвҢҘгҖҚпјҢеҺҹжң¬зҡ„гҖҢзІҳиҙҙйЎ№зӣ®гҖҚе°ұеҸҳжҲҗдәҶгҖҢе°ҶйЎ№зӣ®з§»еҠЁеҲ°иҝҷйҮҢгҖҚ

    • Command+Delete

  • Command+Option+c еӨҚеҲ¶ж–Үд»¶и·Ҝеҫ„
  • жҢүдҪҸ Option еҫҲеӨҡиҸңеҚ•дјҡеҸ‘з”ҹдәӣе°ҸеҸҳеҢ–пјҢжҜ”еҰӮе·ҰдёҠи§’иӢ№жһңеӣҫж ҮзӮ№ејҖпјҢжҢүдҪҸ Option гҖҺе…ідәҺжң¬жңәгҖҸдјҡеҸҳжҲҗгҖҺзі»з»ҹдҝЎжҒҜгҖҸ

и®ҫзҪ®е…Ғи®ёгҖҢд»»дҪ•жқҘжәҗгҖҚзҡ„еә”з”Ё вҖ”вҖ” и§ЈеҶіж–Үд»¶е·ІжҚҹеқҸжҠҘй”ҷ

жңүж—¶еҖҷдјҡйҒҮеҲ°иҝҷз§Қй”ҷиҜҜгҖҢжҠҘй”ҷпјҡж— жі•жү“ејҖе·ІжҚҹеқҸзҡ„е®үиЈ…еҢ…гҖҚпјҢжү“ејҖз»Ҳз«ҜпјҢй”®е…Ҙе‘Ҫд»ӨпјҢиҫ“е…ҘеҜҶз ҒпјҢз„¶еҗҺеӣһиҪҰпјҡ

жү“ејҖгҖҢе®үе…ЁжҖ§дёҺйҡҗз§ҒгҖҚпјҢеҸ‘зҺ°д№…иҝқзҡ„гҖҢд»»дҪ•жқҘжәҗгҖҚеӣһжқҘдәҶгҖӮ

ж”№й”®

дёәдәҶе°ҶеӨ–зҪ®й”®зӣҳе’Ң Mac й”®зӣҳзҡ„ option й”®гҖҒcommand й”®дҝқжҢҒдёҖиҮҙзҡ„йЎәеәҸпјҢеҒҡеҰӮдёӢдҝ®ж”№пјҡ
и®ҫзҪ®-й”®зӣҳ-дҝ®йҘ°й”®-йҖүжӢ©й”®зӣҳпјҲйҖүжӢ©еӨ–и®ҫй”®зӣҳпјү-и°ғж•ҙйЎәеәҸ
